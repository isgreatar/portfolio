Introduction

JavaScript is a cross-platform, object-oriented scripting language used to make webpages interactive and dynamic. It's the language that brings websites to life!

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements that make it incredibly versatile.

Client-side JavaScript extends the core language by supplying objects to control a browser and its Document Object Model (DOM), enabling rich user interactions.

JavaScript and Java

Despite the similar names, JavaScript and Java are completely different programming languages with distinct purposes and characteristics.

The JavaScript language resembles Java syntactically but does not have Java's static typing and strong type checking. JavaScript is dynamically typed and more flexible.

Java is primarily used for large-scale applications, while JavaScript excels in web development and rapid prototyping.

Dynamic Typing Flexible Web-First

Variables

Variables are containers that store data values. Think of them as labeled boxes where you can keep different types of information.

In JavaScript, variable names (identifiers) must follow specific rules but offer great flexibility in naming.

A JavaScript identifier must start with a letter, underscore (_), or dollar sign ($). Subsequent characters can also be digits (0-9).

let const var
// Modern JavaScript variable declarations // let - for variables that can change let userName = "Alice"; let userAge = 25; let isLoggedIn = false; // const - for constants that won't change const PI = 3.14159; const API_URL = "https://api.example.com"; const COLORS = ["red", "green", "blue"]; // Variables can be reassigned (if declared with let) userName = "Bob"; // ‚úÖ This works userAge = 26; // ‚úÖ This works // PI = 3.14; // ‚ùå This would cause an error console.log(`User: ${userName}, Age: ${userAge}`);
// Different data types in variables let message = "Hello, JavaScript!"; // String let count = 42; // Number let price = 19.99; // Number (decimal) let isActive = true; // Boolean let data = null; // Null let result; // Undefined // Objects and Arrays let person = { name: "Sarah", age: 28, city: "New York" }; let fruits = ["apple", "banana", "orange"]; // Check variable types console.log(typeof message); // "string" console.log(typeof count); // "number" console.log(typeof isActive); // "boolean" console.log(typeof person); // "object"
  • let - Use for variables that can be reassigned and have block scope
  • const - Use for constants that won't change after declaration
  • var - Older way of declaring variables (function-scoped, avoid in modern code)
  • Always use meaningful names like userName instead of x
  • Use camelCase for multi-word variable names
